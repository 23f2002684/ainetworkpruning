# Self-Pruning Neural Network Report

## 1. The L1 Penalty and Sparsity on Sigmoid Gates

In the context of the custom `PrunableLinear` layer, an L1 penalty on the sigmoid-activated gate values works as an aggressive sparsity-inducing regularizer. 

The L1 norm penalizes the sum of the absolute values of the weights (or gates, in this case). Since our gate values are constrained between $0$ and $1$ by the sigmoid function, they are strictly positive, making the L1 penalty exactly equal to the sum of the gate values across the network. 

During backpropagation, the L1 penalty produces a constant negative gradient on the gate values (regardless of how close they are to 0), pushing them steadily toward exactly zero. Unlike L2 regularization (weight decay), which pushes weights toward 0 but exerts less force as they get smaller, L1 keeps pushing with the same strength until the gate shuts off. 
When the gate value approaches 0, the corresponding connection is effectively mathematically removed, rendering the network sparse. 

By varying the hyperparameter $\lambda$, we control the balance between the regular classification loss (e.g., Cross-Entropy optimizing for accuracy) and the penalty on active nodes, forcing the network to dynamically trade-off between retaining weights and aggressively dropping them.
